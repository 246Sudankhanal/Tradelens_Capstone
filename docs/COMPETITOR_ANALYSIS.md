# Competitor Analysis - TradeLens

TradeLens targets the gap between manual spreadsheets and expensive cloud products. Below is an overview of the competitive landscape for personal trading journals.

| Competitor | Strengths | Weaknesses / Gap for TradeLens |
| :--- | :--- | :--- |
| **Tradervue** | Established web-based journal, automated import support, detailed statistics. | Subscription limits, third-party cloud storage, dated interface and limited value for cost-sensitive beginners. |
| **Edgewonk** | Detailed behavioural analysis, risk tracking and one-time licence model. | Desktop-focused, complex for beginners and still financially inaccessible to some students. |
| **TraderSync** | Modern UI, mobile access, broker imports and AI-style coaching features. | Premium pricing, cloud dependency and feature complexity for users who only need reliable journaling and analytics. |
| **Excel / Google Sheets** | Free, familiar, flexible and under user control. | Manual calculations, weak validation, no authentication, limited dashboards and poor scalability. |

The key market gap is a free and self-hosted trading journal that combines secure account access, structured CRUD trade management, automatic P&L calculation, CSV import, search/filter/sort tools and meaningful dashboard analytics without requiring a paid subscription or external cloud storage.
